SCRAPEAR EL VENDEDOR DE CHRONO24
================================

URL usada para la prueba:

    https://www.chrono24.com/search/index.htm?customerId=23766&dosearch=true

Para generar el CSV directamente desde terminal, entra a la carpeta principal y ejecuta:

En Windows PowerShell:

    .venv\Scripts\Activate.ps1
    python scraper_chrono24.py "https://www.chrono24.com/search/index.htm?customerId=23766&dosearch=true"

En Windows CMD:

    .venv\Scripts\activate.bat
    python scraper_chrono24.py "https://www.chrono24.com/search/index.htm?customerId=23766&dosearch=true"

En macOS o Linux:

    source .venv/bin/activate
    python scraper_chrono24.py "https://www.chrono24.com/search/index.htm?customerId=23766&dosearch=true"

INSTALAR DEPENDENCIAS SI NO EXISTE .venv
========================================

Si la carpeta .venv no existe, crear el entorno primero.

Windows PowerShell:

    py -m venv .venv
    .venv\Scripts\Activate.ps1
    python -m pip install --upgrade pip
    python -m pip install -r requirements-test.txt

Windows CMD:

    py -m venv .venv
    .venv\Scripts\activate.bat
    python -m pip install --upgrade pip
    python -m pip install -r requirements-test.txt

macOS o Linux:

    python3 -m venv .venv
    source .venv/bin/activate
    python -m pip install --upgrade pip
    python -m pip install -r requirements-test.txt


NOTAS IMPORTANTES
=================

1. Este scraper de Chrono24 no usa credenciales ni APIs.
2. No se necesita Google Sheets para esta prueba.
3. No se necesita token de eBay para esta prueba.
4. Chrono24 puede bloquear requests simples; por eso el scraper usa navegador Selenium cuando hace falta.
5. En Windows y macOS debe estar instalado Google Chrome.
6. Si ChromeDriver falla por version, actualizar Google Chrome normalmente soluciona el problema.
7. El CSV se sobrescribe cada vez que se ejecuta scraper_chrono24.py.
